## Câu hỏi 3

### 📋 NỘI DUNG CÂU HỎI

> A solutions architect is designing an infrastructure for a serverless application. The application is packaged as a Docker image stored in Amazon ECR and must be deployed on a fully managed serverless compute service. Additionally, the application requires 5 GB of ephemeral storage for temporary data processing.

Ứng dụng đóng gói dưới dạng **Docker image** lưu trên ECR, cần triển khai trên dịch vụ **serverless hoàn toàn được quản lý**, và yêu cầu **5 GB ephemeral storage** cho xử lý dữ liệu tạm thời.

---

### 🔑 Từ khóa quyết định đáp án

> "Docker image" + "fully managed serverless" + "5 GB ephemeral storage" (tạm thời, không cần lưu lâu dài)

→ Đây là bài toán về **AWS Lambda với Container Image Support** — vừa serverless, vừa hỗ trợ Docker, vừa cho phép cấu hình ephemeral storage lên đến 10 GB.

---

### 🔍 PHÂN TÍCH TỪNG ĐÁP ÁN

**✅ Đáp án đúng:** Deploy the application in an AWS Lambda function with Container image support. Set the function's storage to 5 GB.

- Lambda hỗ trợ triển khai Docker image — đáp ứng yêu cầu về container.
- Lambda là dịch vụ **fully managed serverless** — không cần quản lý server, tự động scale.
- Ephemeral storage của Lambda có thể cấu hình từ **512 MB đến 10 GB** — đủ để đặt 5 GB.
- Ephemeral storage là tạm thời (wipe sạch sau mỗi lần thực thi) — đúng với yêu cầu "temporary data processing".

---

**❌ Đáp án sai 1:** Deploy the application in an AWS Lambda function with Container image support. Attach an Amazon EFS volume.

- EFS là **persistent storage** (lưu trữ lâu dài, chia sẻ giữa nhiều invocation).
- Đề bài chỉ cần **ephemeral storage** (tạm thời) — EFS là over-engineering, không cần thiết.
- Gắn EFS vào Lambda làm tăng độ phức tạp và chi phí mà không giải quyết đúng yêu cầu.

---

**❌ Đáp án sai 2:** Deploy the application to an Amazon ECS cluster that uses AWS Fargate tasks.

- Fargate cho phép chạy container mà không cần quản lý EC2, nhưng **không phải fully serverless**.
- Vẫn cần cấu hình ECS service, task definition, cluster — tức là vẫn có overhead quản lý hạ tầng.
- Fargate phù hợp cho ứng dụng container cần kiểm soát chi tiết hơn về môi trường.

---

**❌ Đáp án sai 3:** Deploy the application to an Amazon ECS cluster with Amazon EC2 worker nodes and attach a 5 GB Amazon EBS volume.

- Sử dụng **EC2 instances** làm worker nodes — **không phải serverless**.
- Đề bài yêu cầu kiến trúc serverless hoàn toàn — EC2 vi phạm điều kiện này ngay từ đầu.

---

### 🎯 KẾT LUẬN & BÀI HỌC

| Từ khóa trong đề | Giải pháp phù hợp |
|---|---|
| Docker image + fully managed serverless | AWS Lambda (Container Image Support) |
| Ephemeral storage (tạm thời) | Lambda ephemeral storage (tối đa 10 GB) |
| Persistent storage, chia sẻ nhiều invocation | Amazon EFS gắn vào Lambda |
| Container, kiểm soát chi tiết hơn | Amazon ECS + Fargate |
| Không serverless, cần EBS | Amazon ECS + EC2 |

💡 Công thức ghi nhớ:
> "Serverless" + "Docker image" + "ephemeral storage" → **AWS Lambda với Container Image Support**, cấu hình storage tối đa 10 GB.
>
> Phân biệt nhanh: **Ephemeral = tạm thời → Lambda storage**. **Persistent = lâu dài → EFS hoặc EBS**. Đừng nhầm lẫn hai loại này khi đọc đề!
