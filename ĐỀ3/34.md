## Câu hỏi 34

### 📋 NỘI DUNG CÂU HỎI

> A call center wants to use AI to extract insights from **audio recordings** (English & Hindi) to assess customer service quality. Must generate **sentiment analysis report in English** for each recording. Solution must work **without maintaining any machine learning model**.

Trung tâm cuộc gọi muốn phân tích **file audio** (tiếng Anh & Hindi) → tạo báo cáo **phân tích cảm xúc bằng tiếng Anh** → **không cần tự duy trì ML model** nào.

**Câu hỏi:** Giải pháp nào đáp ứng yêu cầu?

---

### 🔑 Từ khóa quyết định đáp án

> "audio → text" + "Hindi → English" + "sentiment analysis" + "no ML model to maintain"

→ Cần pipeline 3 bước: **Transcribe** (audio→text) → **Translate** (Hindi→English) → **Comprehend** (sentiment analysis). Tất cả đều fully managed, không cần train model.

---

### 🧠 Kiến thức nền: AWS AI Services

| Dịch vụ | Chức năng | Ví dụ |
|---|---|---|
| **Amazon Transcribe** | **Speech → Text** (audio thành văn bản) | Transcribe cuộc gọi, tạo subtitle |
| **Amazon Translate** | **Dịch ngôn ngữ** | Hindi → English, Tiếng Việt → Anh |
| **Amazon Comprehend** | **NLP: phân tích văn bản** | Sentiment analysis, entity detection |
| Amazon Polly | Text → Speech (ngược lại Transcribe) | Đọc văn bản thành giọng nói |
| Amazon Rekognition | Nhận dạng **hình ảnh/video** | Phát hiện khuôn mặt, object detection |
| Amazon Lex | Xây dựng **chatbot** | Alexa, voice assistant |
| Amazon Kendra | **Enterprise search** | Tìm kiếm tài liệu nội bộ |

---

### 🔍 PHÂN TÍCH TỪNG ĐÁP ÁN

**✅ Đáp án đúng:** Amazon Transcribe → Amazon Translate → Amazon Comprehend

- **Transcribe**: Chuyển audio (English + Hindi) → text. Hỗ trợ nhiều ngôn ngữ.
- **Translate**: Dịch text tiếng Hindi → tiếng Anh.
- **Comprehend**: Phân tích cảm xúc (positive/negative/neutral) từ text tiếng Anh.
- Cả 3 đều **fully managed** — không cần train hay duy trì ML model.

---

**❌ Đáp án sai 1:** Amazon Polly + Amazon Rekognition + SageMaker BlazingText

- **Polly**: Text → Speech (ngược chiều), không thể transcribe audio.
- **Rekognition**: Nhận dạng hình ảnh/video, không dịch ngôn ngữ.
- **SageMaker BlazingText**: Phải tự train và deploy model → **vi phạm điều kiện không maintain ML model**.

---

**❌ Đáp án sai 2:** Amazon Lex + Amazon Translate + SageMaker Clarify

- **Lex**: Xây dựng chatbot, không phải transcribe audio recordings.
- **SageMaker Clarify**: Dùng để giải thích model predictions và phát hiện bias — không phải sentiment analysis.
- SageMaker vẫn cần tự quản lý model.

---

**❌ Đáp án sai 3:** Amazon Comprehend + Amazon Kendra + Amazon Detective

- **Comprehend**: Phân tích text, không chuyển audio thành text.
- **Kendra**: Enterprise search engine, không dịch ngôn ngữ.
- **Detective**: Dịch vụ bảo mật phân tích security events, không phân tích cảm xúc người dùng.

---

### 🎯 KẾT LUẬN & BÀI HỌC

| Bước | Yêu cầu | AWS Service |
|---|---|---|
| 1 | Audio → Text | **Amazon Transcribe** |
| 2 | Hindi → English | **Amazon Translate** |
| 3 | Sentiment analysis | **Amazon Comprehend** |

| Dịch vụ hay bị nhầm | Thực tế làm gì |
|---|---|
| **Polly** | Text → Speech (không phải Speech → Text) |
| **Rekognition** | Image/Video recognition (không phải dịch thuật) |
| **Lex** | Chatbot (không phải transcribe audio) |
| **Kendra** | Enterprise search (không phải dịch thuật) |

💡 **Công thức ghi nhớ:**
> **Audio → Text = Transcribe. Text dịch = Translate. Phân tích cảm xúc = Comprehend.**
>
> "Without maintaining any ML model" → tránh **SageMaker** (phải tự train/deploy) → dùng các **fully managed AI services** của AWS (Transcribe, Translate, Comprehend, Rekognition...).
