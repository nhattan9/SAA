### 📋 NỘI DUNG CÂU HỎI (Tiếng Anh gốc)

> A Solutions Architect of a multinational gaming company develops video games for PS4, Xbox One, and Nintendo Switch consoles, plus a number of mobile games for Android and iOS. Due to the wide range of their products and services, the architect proposed that they use API Gateway.
>
> What are the key features of API Gateway that the architect can tell to the client? (Select TWO.)

---

### 🔤 DỊCH CHI TIẾT

Một Solutions Architect của công ty game đa quốc gia phát triển video game cho PS4, Xbox One, Nintendo Switch và các mobile game cho Android/iOS. Do phạm vi sản phẩm rộng, architect đề xuất sử dụng API Gateway.

Các tính năng chính nào của API Gateway mà architect có thể giới thiệu với client? (Chọn HAI.)

---

### 🎯 PHÂN TÍCH YÊU CẦU CHÍNH

Câu hỏi kiểm tra kiến thức về **đặc tính cốt lõi của Amazon API Gateway** và khả năng phân biệt với các dịch vụ AWS khác hay bị nhầm lẫn (Global Accelerator, Elastic Fabric Adapter).

---

### 🔍 PHÂN TÍCH TỪNG ĐÁP ÁN

✅ **Đáp án đúng 1: Enables you to build RESTful APIs and WebSocket APIs that are optimized for serverless workloads.**

- API Gateway hỗ trợ tạo **REST API** và **WebSocket API** tích hợp trực tiếp với AWS Lambda → tối ưu cho serverless architecture. ✅
- Phù hợp với gaming company cần API cho nhiều nền tảng (console, mobile) mà không cần quản lý server. ✅

---

✅ **Đáp án đúng 2: You pay only for the API calls you receive and the amount of data transferred out.**

- API Gateway theo mô hình **pay-per-use**: không có phí tối thiểu hay startup cost. ✅
- Chỉ tính phí theo số lượng API call nhận được và lượng data transfer out. ✅
- Rất phù hợp với gaming company có traffic biến động theo từng game mới ra mắt. ✅

---

❌ **Đáp án sai 1: It automatically provides a query language for your APIs similar to GraphQL.**

- API Gateway **không cung cấp query language** kiểu GraphQL.
- GraphQL trên AWS được hỗ trợ bởi **AWS AppSync**, không phải API Gateway. ✗

---

❌ **Đáp án sai 2: Enables you to run applications requiring high levels of inter-node communications at scale through its custom-built OS bypass hardware interface.**

- Đây là đặc tính của **Elastic Fabric Adapter (EFA)** — network interface chuyên dụng cho HPC (High Performance Computing) và machine learning workload cần giao tiếp nhanh giữa các node. ✗
- Hoàn toàn không liên quan đến API Gateway. ✗

---

❌ **Đáp án sai 3: Provides you with static anycast IP addresses that serve as a fixed entry point to your applications hosted in one or more AWS Regions.**

- Đây là đặc tính của **AWS Global Accelerator** — cung cấp static anycast IP làm entry point cố định, route traffic tối ưu đến endpoint gần nhất. ✗
- API Gateway không cung cấp static anycast IP. ✗

---

### 📊 KẾT LUẬN & BÀI HỌC

| Tính năng | Dịch vụ đúng |
|---|---|
| Build RESTful API & WebSocket API, serverless | **Amazon API Gateway** |
| Pay-per-use: tính phí theo API call & data out | **Amazon API Gateway** |
| GraphQL query language cho API | **AWS AppSync** |
| Static anycast IP, fixed entry point đa region | **AWS Global Accelerator** |
| OS bypass hardware interface, inter-node HPC | **Elastic Fabric Adapter (EFA)** |

💡 **Công thức ghi nhớ:**

> "RESTful API, WebSocket API, serverless" → **API Gateway**

> "GraphQL, real-time data sync" → **AWS AppSync**

> "Static anycast IP, multi-region entry point" → **AWS Global Accelerator**

> "HPC, inter-node communication, OS bypass" → **Elastic Fabric Adapter (EFA)**

> API Gateway = **no minimum fee**, pay only for calls received + data transferred out.
