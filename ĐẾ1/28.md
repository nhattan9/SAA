# 1. Dịch câu hỏi

Một doanh nghiệp vừa migrate application lên AWS.

Audit team cần đánh giá xem các dịch vụ công ty đang sử dụng có đáp ứng các tiêu chuẩn:

* security
* regulatory compliance

hay không.

Solutions Architect cần cung cấp cho audit team một báo cáo gồm các tài liệu liên quan đến compliance cho AWS account của họ.

Nên thực hiện hành động nào?

---

# 2. Đáp án đúng

✅ **Use AWS Artifact to view the security reports as well as other AWS compliance-related information.**

Dịch:

> Sử dụng AWS Artifact để xem các security reports và các thông tin/tài liệu liên quan đến compliance của AWS.

# 4. AWS Artifact là gì?

AWS Artifact là nơi trung tâm để tải/xem:

* AWS security reports
* compliance reports
* agreements
* certifications

Ví dụ:

* SOC reports
* PCI reports
* ISO certifications
* BAA
* NDA

---

# 5. Vì sao AWS Artifact đúng?

Vì đề yêu cầu:

```text
report of all compliance-related documents
```

AWS Artifact sinh ra để cung cấp đúng loại tài liệu này.

Audit team có thể dùng nó để lấy bằng chứng AWS đáp ứng các tiêu chuẩn bảo mật/tuân thủ.

---

# 6. Vì sao các đáp án khác sai?

## ❌ AWS Security Hub

Security Hub dùng để:

* tổng hợp security findings
* xem security posture
* kiểm tra best practices

Không phải nơi tải compliance reports như SOC/PCI.

---

## ❌ Amazon Inspector

Inspector dùng để:

* scan vulnerability
* đánh giá security của EC2/container/Lambda

Không cung cấp bộ tài liệu compliance của AWS.

---

## ❌ Amazon Macie + ACM

Macie dùng để:

* phát hiện dữ liệu nhạy cảm trong S3
* PII discovery

ACM dùng để:

* quản lý SSL/TLS certificates

Không phải nơi chứa SOC/PCI/compliance reports.

---

# 7. Phân biệt nhanh

| Service      | Dùng để làm gì               |
| ------------ | ---------------------------- |
| AWS Artifact | Compliance reports/documents |
