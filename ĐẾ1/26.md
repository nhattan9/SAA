# 4. DDoS là gì?

# Distributed Denial of Service

Là kiểu attack:

```text id="r0dl3y"
rất nhiều máy cùng gửi traffic cực lớn
```

làm:

* server overload
* nghẽn bandwidth
* application down
* service unavailable

---

# 8. Vì sao các đáp án khác sai?

---

## ❌ AWS WAF

AWS WAF bảo vệ:

# Layer 7 (HTTP/HTTPS)

Ví dụ:

* SQL injection
* XSS
* bad HTTP patterns

---

WAF KHÔNG phải service chính chống:

# large-scale DDoS

Đặc biệt:

* SYN flood
* volumetric attacks
* UDP reflection

=> Shield phù hợp hơn.

---

# 9. WAF vs Shield

| Service    | Chống gì                              |
| ---------- | ------------------------------------- |
| AWS WAF    | Web attacks (SQLi, XSS, bad requests) |
| AWS Shield | DDoS attacks                          |

---

# 10. Vì sao Firewall Manager sai?

Firewall Manager dùng để:

# central management

cho:

* WAF
* Shield
* Security policies

Nó là service quản lý policy.

Không phải DDoS protection engine.

---

# 11. Vì sao Security Group + NACL sai?

Security Group và NACL:

* allow/deny traffic
* filter ports/protocols

Nhưng không chống được:

# volumetric DDoS

Ví dụ:

```text id="01j1g8"
100 triệu requests/second
```

SG/NACL không đủ khả năng absorb attack kiểu đó.
