# 1. Dịch đề

Một công ty e-commerce đang dùng:

# Amazon API Gateway Regional

để host public REST APIs.

API Gateway được truy cập thông qua:

* custom domain name
* Route 53 alias record

Công ty muốn:

* release phiên bản API mới
* có thêm feature mới
* optimize performance

Yêu cầu:

```text
reduce customer disruption
minimal data loss
MOST cost-effective
```

# 4. Canary Release là gì?

Canary deployment:

# release cho một phần nhỏ user trước

Ví dụ:

```text id="s20l5j"
5% traffic → API mới
95% traffic → API cũ
```

---

# Nếu ổn?

Tăng dần:

```text id="q2xnlh"
5% → 20% → 50% → 100%
```

---

# Nếu lỗi?

Rollback rất nhanh.

Chỉ ảnh hưởng một nhóm nhỏ user.

---

# 5. Vì sao đây là đáp án tốt nhất?

---

# ✅ Minimal disruption

Vì:

```text id="nlnh3m"
đa số user vẫn dùng API cũ
```

---

# ✅ Minimal data loss

Nếu API mới lỗi:

* chỉ một phần nhỏ traffic bị ảnh hưởng
* rollback nhanh

---

# ✅ MOST cost-effective

Canary deployment trong API Gateway:

* built-in
* không cần duplicate full infrastructure
* không cần maintain 2 environments lớn

---

# 6. Canary trong API Gateway hoạt động thế nào?

API Gateway hỗ trợ:

# Canary Stage Deployment

Ví dụ:

```text id="71n6jv"
Production Stage
 ├── 90% traffic → old deployment
 └── 10% traffic → canary deployment
```

---

# 7. Vì sao Blue-Green sai?

Blue-Green cũng giảm downtime tốt.

NHƯNG:

```text id="49qcl9"
MOST cost-effective
```

Blue-Green cần:

* maintain 2 full environments
* double infrastructure
* phức tạp hơn

=> cost cao hơn Canary.

---

# So sánh cực quan trọng

| Canary               | Blue-Green               |
| -------------------- | ------------------------ |
| rollout dần dần      | switch toàn bộ           |
| rẻ hơn               | đắt hơn                  |
| ít infra duplication | cần 2 env                |
| phù hợp API Gateway  | dùng nhiều cho infra lớn |

---

# 8. Vì sao tạo API Gateway mới sai?

```text id="1x8o8m"
Create new API Gateway
```

Sai vì:

* phải switch domain
* DNS propagation delay
* có thể downtime
* customer disruption

---

# 9. Vì sao import-to-update sai?

```text id="jlwmu4"
overwrite / merge mode
```

Sai vì:

* update trực tiếp production
* không có traffic isolation
* nếu lỗi → ảnh hưởng toàn bộ user
