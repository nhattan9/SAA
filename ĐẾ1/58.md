# Dịch đề

Một công ty sử dụng:

# Application Load Balancer (ALB)

cho các ứng dụng web multi-tier public-facing.

Đội security vừa phát hiện:

# SQL Injection attacks tăng mạnh

gây ra:

```text
critical data discrepancy issues
```

Ngoài ra các web applications ở:

```text
multiple AWS accounts
```

cũng bị ảnh hưởng tương tự.

Cần có giải pháp ngay lập tức để:

* chặn SQL Injection
* bảo vệ ứng dụng trên nhiều AWS accounts

---

# Đáp án đúng

✅

```text
Use AWS WAF and set up a managed rule to block request patterns associated with SQL injection attacks.
Associate it with the ALB.
Integrate AWS WAF with AWS Firewall Manager across all AWS accounts.
```

Dịch:

> Sử dụng AWS WAF và cấu hình managed rule để chặn các request có pattern của SQL Injection.
> Associate WAF với ALB.
> Tích hợp với AWS Firewall Manager để reuse rules trên nhiều AWS accounts.

# AWS WAF là gì?

AWS WAF:

# Web Application Firewall

Dùng để filter/chặn:

* SQL Injection
* XSS
* bad HTTP requests
* malicious patterns

# SQL Injection được chặn thế nào?

AWS WAF có:

# Managed Rule Groups

Ví dụ:

```text
AWSManagedRulesSQLiRuleSet
```

Rule này detect pattern như:

```sql
' OR 1=1 --
UNION SELECT
DROP TABLE
```

=> block request trước khi vào app.
