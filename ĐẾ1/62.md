# Dịch đề

Một công ty retail nhận các file dữ liệu thô `.csv` vào Amazon S3 bucket từ nhiều nguồn khác nhau theo mỗi giờ.

Thông tin:

* file size trung bình: `2 GB`
* bucket được quản lý bằng:

```text id="j7dxq6"
S3 Access Grants
```

để fine-grained access control.

Yêu cầu:

* tự động convert `.csv`
  → thành:

# Apache Parquet

* lưu file output vào S3 bucket khác
* conversion process phải:

# tự động trigger ngay khi có file mới upload

* đồng thời:

# LEAST operational overhead

---

# Đáp án đúng

✅

```text id="57gq1u"
Use AWS Glue ETL job + EventBridge trigger on S3 Object Created events
```

Dịch:

> Sử dụng AWS Glue ETL job để xử lý và convert file `.csv` sang Apache Parquet rồi lưu vào target S3 bucket.
> Dùng Amazon EventBridge để trigger Glue job khi có S3 Object Created event.

# Vì sao dùng AWS Glue?

AWS Glue là:

# serverless ETL service

Rất phù hợp để:

* xử lý big data
* transform dữ liệu
* convert format

Ví dụ:

```text id="6j3yl1"
CSV
→ Parquet
```

---

# Vì sao Parquet quan trọng?

Parquet là:

# columnar storage format

Ưu điểm:

✅ query nhanh hơn
✅ compress tốt hơn
✅ tối ưu big data analytics


# Vì sao dùng EventBridge?

Đề yêu cầu:


# Luồng đúng

```text id="prgl93"
New CSV upload to S3
        ↓
S3 Object Created Event
        ↓
EventBridge Rule
        ↓
AWS Glue ETL Job
        ↓
Convert CSV → Parquet
        ↓
Store into target S3 bucket
```

# Vì sao các đáp án khác sai?

---

# ❌ Spark on EC2

Sai vì:

* phải tự quản EC2
* manage Spark cluster
* scaling
* monitoring
* maintenance

=> operational overhead cao.

---

# ❌ Glue crawler scheduled every hour

Sai vì:

Đề yêu cầu:

```text id="k4dh1s"
trigger ngay khi upload
```

Scheduled crawler:

```text id="5mj0nw"
delay up to 1 hour
```

Không event-driven realtime.

---

# ❌ Lambda convert CSV → Parquet

Sai vì:

File size:

```text id="y4tr92"
2 GB
```

Lambda không phù hợp big data processing.

---

# Lambda limitations

| Limitation        | Problem     |
| ----------------- | ----------- |
| Timeout           | max 15 phút |
| Memory            | limited     |
| Ephemeral storage | limited     |

CSV 2 GB convert sang Parquet dễ timeout/memory issue.

---

# AWS Transfer Family hoàn toàn không cần thiết

Transfer Family dùng cho:

```text id="5y3ms0"
SFTP/FTP/FTPS
```

Không liên quan conversion workflow.
