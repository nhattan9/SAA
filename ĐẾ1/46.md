# Dịch đề

Một ứng dụng Forex trading dùng AI cần tiêu thụ hàng nghìn bộ dữ liệu để train machine learning model.

Workload của ứng dụng cần:

* **hot storage hiệu năng cao**
* hỗ trợ **parallel processing**
* xử lý nhiều training datasets cùng lúc

Ngoài ra, ứng dụng cũng cần:

* **cold storage tiết kiệm chi phí**
* để archive các dataset có lợi nhuận thấp, ít dùng lại

Hỏi developer nên dùng dịch vụ storage nào?

---

# Đáp án đúng

✅

```text
Use Amazon FSx For Lustre and Amazon S3 for hot and cold storage respectively.
```

Dịch:

> Dùng Amazon FSx for Lustre cho hot storage và Amazon S3 cho cold storage.

# Vì sao hot storage là FSx for Lustre?

**Amazon FSx for Lustre** được thiết kế cho workload cần hiệu năng cao như:

* machine learning
* high performance computing
* big data analytics
* media processing
* financial modeling

Lustre là **parallel file system**.

Tức là nhiều máy/tiến trình có thể đọc ghi dữ liệu song song với throughput rất cao.

Luồng phù hợp:

```text
Training datasets
→ FSx for Lustre
→ ML application xử lý song song
```

---

# Vì sao cold storage là S3?

Amazon S3 phù hợp lưu trữ lâu dài, rẻ, bền vững.

Với dữ liệu archive, có thể dùng:

```text
S3 Glacier
S3 Glacier Deep Archive
```

Phù hợp cho:

* dữ liệu ít truy cập
* backup
* archive
* lưu lâu dài với chi phí thấp

---

# Vì sao các đáp án khác sai?

## ❌ FSx for Windows File Server + S3

Sai phần hot storage.

FSx for Windows File Server dùng cho:

```text
Windows file share
SMB protocol
Active Directory
```

Nó không phải parallel file system cho ML/HPC.

---

## ❌ EFS + S3

EFS có thể share file cho nhiều instance, nhưng không tối ưu bằng FSx for Lustre cho workload:

```text
high-performance parallel processing
machine learning training
```

EFS hợp hơn cho web app, CMS, shared files.

---

## ❌ FSx for Lustre + EBS io1

Sai phần cold storage.

EBS io1 là:

```text
Provisioned IOPS SSD
```

Dùng cho workload cần I/O cao, dữ liệu nóng.

Nó đắt, không phù hợp archive cold data.

Cold archive rẻ hơn nên dùng:

```text
S3 Glacier / Glacier Deep Archive
```

---

# Bảng nhớ nhanh

| Requirement                            | Dịch vụ phù hợp             |
| -------------------------------------- | --------------------------- |
| ML training, HPC, parallel file system | FSx for Lustre              |
| Windows shared file SMB                | FSx for Windows File Server |
| Linux shared file NFS                  | EFS                         |
| Object storage/archive/cold data       | S3 / Glacier                |
| High IOPS block storage                | EBS io1/io2                 |
