## Câu hỏi 35

---

### 📋 NỘI DUNG CÂU HỎI (Tiếng Anh gốc)

> An airline company receives a lot of requests to book flights, update booking details, and flight check-ins. Since these requests flood the customer support teams, the management wants to build a self-service solution that can handle these requests without a human agent. This solution should be text-based, allowing users to type concerns in a chat box while AI analyzes intentions, provides answers, or fulfills predefined actions automatically.
>
> Which of the following options is the recommended solution for the above requirements?

---

### 🔤 DỊCH CHI TIẾT

Công ty hàng không nhận rất nhiều yêu cầu đặt vé, cập nhật booking, và check-in. Để giảm tải cho nhân viên, công ty muốn xây dựng **self-service solution** (giải pháp tự phục vụ) **không cần human agent**. Giải pháp phải:
- **Text-based** — người dùng gõ vào chat box.
- **AI phân tích intent** (ý định của người dùng).
- **Tự động thực hiện hành động** tương ứng (đặt vé, cập nhật, check-in...).

---

### 🎯 PHÂN TÍCH YÊU CẦU CHÍNH

1. **Conversational chatbot** — AI hiểu ngôn ngữ tự nhiên, nhận dạng intent của người dùng.
2. **Text-based** — input là text, không phải giọng nói hay hình ảnh.
3. **Thực hiện hành động tự động** — cần tích hợp với backend logic (đặt vé, cập nhật DB...).

> ⚠️ Điểm dễ nhầm: Comprehend làm NLP nhưng **không build được chatbot**. Polly chuyển text → speech, không phải chatbot. Rekognition phân tích hình ảnh, không liên quan chatbot.

---

### 🔍 PHÂN TÍCH TỪNG ĐÁP ÁN

---

✅ **Đáp án đúng (B):** Deploy a conversational chatbot using **Amazon Lex** + **AWS Lambda** code hooks

- **Amazon Lex** là service chuyên build **conversational chatbot** — cùng công nghệ với Amazon Alexa, hỗ trợ **NLU** (Natural Language Understanding) và nhận dạng **intent** từ text input. ✅
- Cho phép định nghĩa **conversation flow** cho từng intent (đặt vé, check-in, cập nhật...). ✅
- Tích hợp **Lambda functions làm code hooks** → thực hiện logic backend tự động khi user có intent cụ thể. ✅

---

❌ **Đáp án sai (C):** Create a conversational chatbot using **Amazon Comprehend** + Lambda

- **Amazon Comprehend** là NLP service — phân tích sentiment, entity, key phrase trong văn bản có sẵn.
- **Không có khả năng build chatbot** — không quản lý conversation flow, không tương tác real-time với người dùng.

---

❌ **Đáp án sai (D):** **Amazon Polly** + SSML + AWS MSP + Lambda

- **Amazon Polly** là service **text-to-speech** (chuyển văn bản → giọng nói) — không phải chatbot, không hiểu intent.
- Đề bài yêu cầu **text-based**, không cần speech synthesis.
- **SSML** là ngôn ngữ định dạng giọng nói → không liên quan đến text chatbot.

---

❌ **Đáp án sai (A):** Deploy a conversational chatbot using **Amazon Rekognition** + Lambda

- **Amazon Rekognition** phân tích **hình ảnh và video** (nhận dạng khuôn mặt, object, scene...).
- Hoàn toàn không liên quan đến text chatbot hay NLP.

---

### Note / Pipeline đúng

Yêu cầu: Text-based chatbot tự phục vụ, nhận dạng intent, thực hiện hành động tự động.

```
User gõ text vào chat box
→ Amazon Lex nhận text input
→ NLU engine phân tích → nhận dạng Intent
   (VD: "BookFlight", "UpdateBooking", "CheckIn")
→ Lex trigger Lambda function tương ứng (code hook)
→ Lambda thực hiện logic backend
   (tra cứu chuyến bay, cập nhật DB, xác nhận check-in...)
→ Lex trả kết quả về chat box cho user
(Không cần human agent)
```

### 📊 KẾT LUẬN & BÀI HỌC

| AWS AI Service | Chức năng thực sự | Dùng khi nào |
|---|---|---|
| **Amazon Lex** | Build conversational chatbot, nhận dạng intent | Chatbot text/voice |
| **Amazon Comprehend** | Phân tích NLP: sentiment, entity, key phrase | Phân tích văn bản có sẵn |
| **Amazon Polly** | Text → Speech (giọng nói tổng hợp) | Text-to-speech app |
| **Amazon Rekognition** | Phân tích hình ảnh/video | Computer vision |
| **Amazon Transcribe** | Speech → Text | Speech-to-text |

💡 **Công thức ghi nhớ:**

> **"Chatbot + intent + conversation flow"** → **Amazon Lex**

> **"Phân tích cảm xúc/thực thể trong văn bản"** → **Amazon Comprehend** (không phải chatbot)

> **"Text → Giọng nói"** → **Amazon Polly**

> **"Giọng nói → Text"** → **Amazon Transcribe**

> **"Hình ảnh / Video"** → **Amazon Rekognition**
