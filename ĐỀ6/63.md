## Câu hỏi 64

### 📋 NỘI DUNG CÂU HỎI (Tiếng Anh gốc)

> A **Solutions Architect** is trying to **enable Cross-Region Replication** to an **Amazon S3 bucket** but **this option is disabled**. The **bucket has been configured with a Multi-Region Access Point** to manage incoming requests.
>
> Which of the following options is a **valid reason** for this?

---

### 🔤 DỊCH CHI TIẾT

Một **Solutions Architect** (kiến trúc sư giải pháp) đang cố gắng **enable Cross-Region Replication** (bật tính năng sao chép giữa các region) đến **Amazon S3 bucket** (S3 bucket) nhưng **this option is disabled** (tùy chọn này bị vô hiệu hóa). **Bucket has been configured with a Multi-Region Access Point** (bucket được cấu hình với Multi-Region Access Point - điểm truy cập đa region) để **manage incoming requests** (quản lý các yêu cầu đến). **Which of the following options is a valid reason for this?** (Tùy chọn nào sau đây là một lý do hợp lệ cho điều này?)

---

### 🎯 PHÂN TÍCH YÀU CẦU CHÍNH

1. **Cross-Region Replication (CRR)** disabled
2. **Multi-Region Access Point** configured
3. **Why is CRR disabled?** → Tìm lý do

---

### 🔍 PHÂN TÍCH TỪNG ĐÁP ÁN

❌ **A. This is a premium feature which is only for AWS Enterprise accounts**

- ❌ CRR **KHÔNG phải premium feature**
- ❌ Có sẵn cho tất cả support plans
- ❌ Sai hoàn toàn

✅ **B. In order to use Cross-Region Replication, you need to first enable versioning on the bucket**

- ✅ **Versioning là requirement**
- ✅ Source bucket cần enable versioning
- ✅ Destination bucket cần enable versioning
- ✅ KHÔNG có versioning → CRR disabled

❌ **C. Cross-Region Replication feature only available for S3 - Infrequent Access**

- ❌ CRR available cho **tất cả storage classes**
- ❌ Standard, Standard-IA, One Zone-IA, Glacier, ... đều support

❌ **D. Cross-Region Replication feature only available for S3 - One Zone-IA**

- ❌ CRR available cho **tất cả storage classes**
- ❌ KHÔNG giới hạn storage class

---

**📌 Note — Prerequisites for Cross-Region Replication:**

```
Requirements để enable CRR:

1️⃣ Source bucket
     ├─ ✅ Versioning ENABLED (bắt buộc)
     └─ ❌ KHÔNG versioning → CRR disabled
     
2️⃣ Destination bucket
     ├─ ✅ Versioning ENABLED (bắt buộc)
     └─ ❌ KHÔNG versioning → CRR disabled
     
3️⃣ Buckets ở khác nhau
     ├─ ✅ Source: us-east-1
     └─ ✅ Destination: eu-west-1
     
4️⃣ IAM permissions
     ├─ ✅ S3 phải có permission replicate objects
     └─ ❌ KHÔNG permission → CRR fails
```

### 📊 KẾT LUẬN & BÀI HỌC

| Requirement | Status | CRR enabled? |
|---|---|---|
| **Versioning enabled** | ✅ Yes | ✅ Yes |
| **Versioning enabled** | ❌ No | ❌ **No (disabled)** |
| Different regions | ✅ Yes | ✅ Yes |
| IAM permissions | ✅ Yes | ✅ Yes |

| Reason | Valid? |
|---|---|
| **Versioning not enabled** | ✅ **Yes** |
| Premium feature | ❌ No |
| Only for Standard-IA | ❌ No |
| Only for One Zone-IA | ❌ No |
| Storage class limitation | ❌ No |

> 💡 **Công thức ghi nhớ:**
> **"CRR disabled? → Check versioning first!"**
> **"Cross-Region Replication = Versioning + Different regions + IAM permission"**

> 🔑 **Keyword bẫy:**
> ✅ **"Versioning"** → **MUST enable** cho source & destination
> ❌ **"Premium feature"** → Available to all, not premium
> ❌ **"Storage class specific"** → Available for all classes
> ✅ **"Multi-Region Access Point"** → Cấu hình access, KHÔNG affect versioning requirement
